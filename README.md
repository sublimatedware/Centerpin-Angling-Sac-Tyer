# Centerpin Angling Sac Tyer

This is a work-in-progress device to automatically tie roe spawn sacs for recreational fishing. Mechanical aspects of the device are primarily 3D printed with some use of aluminum extrusions and modified consumer electronics. It is controlled by an STM32 F411CEU6 "blackpill" and uses a windows client written in python for setup, monitoring, logging, and eventually more advanced computer vision driven control.
